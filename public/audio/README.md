Place licensed background music files here.

The Iceland garden is wired to:

`/audio/akureyri-bing-yu.mp3`

Optional synced lyric text can be supplied at:

`/audio/akureyri-bing-yu.lrc`

I added `akureyri-bing-yu.lrc.template` with the timestamp slots. Duplicate it as
`akureyri-bing-yu.lrc` and fill in lyric text only if you have the rights to use
that copy.

The app does not bundle copyrighted audio or lyrics. Use a file you have rights
to publish, or replace the track/YouTube settings in `app/page.tsx`.
